These files need to be placed in the SODAQ SAMD boards hardware folder.
They replace existing files.

On Windows this folder can be found at:
/Users/<USERNAME>/AppData/Roaming/Arduino15/packages/SODAQ/hardware/samd/1.6.3

boards.txt  = Root folder at the above location

variant.h   = In the ./variants/sodaq_autonomo/ subfolder of the above location
variant.cpp = In the ./variants/sodaq_autonomo/ subfolder of the above location
